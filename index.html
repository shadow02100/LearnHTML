How to start Coding in HTML

First off you are going to want to open your text editor of choice, for me I'm using
Sublime Text 3

First off we need to know how to open and close tags. In HTML opening tags are denoted by ""
a Less Than Sign to open the tag, and a Greater Than Sign to close it off. Closing tags are denoted
by this "", notice the foward slash in there, that's what declares it as a closing tag.

Now that we know how to open and close tags, we can start coding!

In that text editor you are going to want to create two HTML tags, one opening
and one closing tag and save the file *anything*.html (you can change the anything to
what ever you want).

These two tags tell the browser that we are in fact using HTML

But we're not done yet, that is just the start of it all.
